🔗 Supabase 接続設定
無料でリアルタイム共有が可能になります。supabase.com で Project URL と anon key を取得してください。
① Supabase SQL Editor で実行するSQL
-- クライアントテーブル create table if not exists clients ( id text primary key, kind text not null, company text not null, industry text, scale text, branches text, contact text, ctitle text, phone text, email text, area text, addr text, web text, assignee text, stage text, memo text, created_at text ); -- アクションテーブル create table if not exists actions ( id text primary key, client_id text references clients(id) on delete cascade, type text, note text, rep text, date text ); -- リアルタイム有効化 alter publication supabase_realtime add table clients; alter publication supabase_realtime add table actions;
手順：
1. supabase.com → 無料アカウント作成
2. New Project → リージョン「Northeast Asia (Tokyo)」
3. 左メニュー「SQL Editor」→ 上のSQLを貼り付けて実行
4. 「Settings → API」から URL と anon key をコピーして上に入力
5. 「接続テスト」→ 成功したら「保存して接続」
